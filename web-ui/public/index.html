Setup Guide
How to use this PXE Boot Server:
  1. Connect your mini PC to the same network as this server
  2. Configure your mini PC's BIOS to boot from network (PXE)
  3. Restart your mini PC and it should receive a DHCP address and boot menu
  4. Select "Install Ubuntu 22.04 LTS" from the boot menu
  5. The installation will proceed automatically using the settings configured above
Troubleshooting:
  • Ensure all services are running (green status indicators above)
  • Check that your mini PC supports PXE booting
  • Make sure the network connection is stable
  • If the installation fails, check the logs in the container
